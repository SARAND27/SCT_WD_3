# Tic Tac Toe Web Application

## SkillCraft Technology Internship - Task 3

This project is a Tic Tac Toe Web Application developed as part of the SkillCraft Technology Web Development Internship.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Two-Player Gameplay
* Winner Detection
* Draw Detection
* Restart Game Functionality
* Responsive Design

## How to Play

1. Player X starts the game.
2. Players take turns marking empty cells.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
4. If all cells are filled and no player wins, the game ends in a draw.
